Administration

Publiez articles, événements, exercices et médias directement dans le dépôt GitHub. Chaque enregistrement crée un commit sur main, ce qui déclenche automatiquement la compilation LaTeX et le redéploiement du site (2 à 5 minutes).

⚠️ Sécurité du jeton. Ce panneau fonctionne sans serveur : votre Personal Access Token GitHub est stocké uniquement dans le localStorage de ce navigateur et n'est transmis qu'à api.github.com. Un jeton avec accès en écriture au dépôt permet d'en modifier tout le contenu : utilisez un jeton « fine-grained » limité à ce seul dépôt (permission Contents : Read and write), avec une date d'expiration courte, et uniquement sur un appareil de confiance. Cliquez sur « Oublier le jeton » après usage sur un poste partagé.
