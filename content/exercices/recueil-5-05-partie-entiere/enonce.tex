Soient $x, y$ deux nombres réels et $n$ un entier naturel non nul.
\begin{itemize}
  \item Montrer : $E(x) + E(y) \leq E(x + y)$.
  \item Montrer : $\lfloor \lfloor x \rfloor / n \rfloor = \lfloor x/n \rfloor$.
  \item $\lfloor x \rfloor + \lfloor y \rfloor + \lfloor x + y \rfloor \leq \lfloor 2x \rfloor + \lfloor 2y \rfloor$.
  \item Calculer $\left\lfloor \sqrt{1} \right\rfloor + \left\lfloor \sqrt{2} \right\rfloor + \cdots + \left\lfloor \sqrt{2026} \right\rfloor$.
\end{itemize}
