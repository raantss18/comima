On définit une fonction $T$ ainsi :
\[
T(n) =
\begin{cases}
\dfrac{n}{2}, & \text{si } n \text{ est pair,}\\
3n + 1, & \text{si } n \text{ est impair.}
\end{cases}
\]

La \textbf{conjecture de Collatz} dit ceci : Pour tout entier $n > 1$, si on
applique plusieurs fois $T$, la suite obtenue arrive toujours à $1$, puis
continue dans le cycle :
\[
1 \ \to\ 4 \ \to\ 2 \ \to\ 1 \ \to\ \cdots
\]
