Une matrice $n \times n$ dont les coefficients appartiennent à l'ensemble $S = \{1, 2, \dots, 2n-1\}$ est dite \emph{argentée} si, pour chaque $i = 1, 2, \dots, n$, la $i$-ème ligne et la $i$-ème colonne contiennent ensemble tous les éléments de $S$. Montrer que :
\begin{enumerate}
\item il n'existe pas de matrice argentée pour $n = 1997$ ;
\item il existe des matrices argentées pour une infinité de valeurs de $n$.
\end{enumerate}
